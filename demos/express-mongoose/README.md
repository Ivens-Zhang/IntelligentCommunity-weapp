## 一个学生CRUD系统

### 运用到的技术:

- art-template
- express
- mongoose

### 文件结构
```cmd
E:.
│  node_modules
│  app.js
│  db.js
│  package.json
│  README.md
│  router.js
│
├─public
│  └─css
│          dashboard.css
│
└─views
        edit.html
        index.html
        new.html
```

### 使用方法

输入:

```cmd
$ npm install
$ node app.js
```
